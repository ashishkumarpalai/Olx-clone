# Market Zone

> This website is a clone of "OLX" E-commers website . The OLX marketplace is a platform for buying and selling services and goods such as electronics, fashion items, furniture, household goods, cars and bikes.

## TechStack used

### Frontend
+ HTML
+ CSS
+ JAVASCRIPT

### Backend
+ mongoDb Atlash
+ nodejs
+ express

### Content of Website
+ User Login
+ Admin Login 
+ Home Page
+ Career Page
+ Menu/Product Page
+ Cart Page
+ Payment Page
+ Admin Page
